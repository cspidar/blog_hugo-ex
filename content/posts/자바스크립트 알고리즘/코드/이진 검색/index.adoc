== 이진 검색

* lt: 맨 왼쪽 / rt: 맨 오른쪽 / mid: lt와 rt의 중간
* arr의 mid 값이 condition() 함수의 조건 만족할때까지 검색

[source,js]
----
while (lt <= rt) {
  mid = Math.floor((lt + rt) / 2);
  if (condition(arr, mid) === false) {
    res = mid;
    lt = mid + 1; // mid는 이미 확인했기때문에 rt => mid - 1 / lt => mid + 1
  } else rt = mid - 1;
}
----
